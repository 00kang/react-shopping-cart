# 페이먼츠

## 기능 목록

- 내비게이션 바
  - [ ] 로고를 클릭하면 홈으로 이동할 수 있다.
  - [ ] 장바구니를 클릭하면 장바구니 페이지로 이동할 수 있다.
  - [ ] 주문 목록을 클릭하면 주문 목록 페이지로 이동할 수 있다.
- 상품 목록
  - [x] 상품을 확인할 수 있다.
    - [x] 상품 이미지
    - [x] 상품 가격
    - [x] 장바구니 버튼
  - [ ] 장바구니 버튼을 클릭하면 상품을 장바구니에 담을 수 있다.
  - [ ] 반응형 레이아웃을 확인할 수 있다.
  - [ ] 최초 검색 결과는 12개까지만 보여준다. 스크롤을 내려 추가 검색 결과를 불러올 수 있다.
  - [x] 상품목록을 server에서 불러올 수 있다.
